# 🎲 Dice Game

A simple and fun **two-player dice game** built using **HTML, CSS, and JavaScript**.  
The game simulates a dice roll and lets two players compete by tracking their scores until one reaches the winning score.

---

## ✨ Features
- 🎲 Random dice rolls using `Math.random()`
- 👥 Two-player gameplay with score tracking
- 🔄 Automatic switching between players
- 🏆 Declares winner when the target score is reached
- 🔁 Reset button to restart the game

---

## 🛠️ Technologies Used
- **HTML** – structure  
- **CSS** – styling and layout  
- **JavaScript** – game logic, event handling, and DOM manipulation  

---

## 📖 How to Play
1. Open the game in your browser.  
2. Player 1 starts by rolling the dice.  
3. Each roll adds points to the current score.  
4. If a player rolls a **1**, they lose their turn, and the next player plays.  
5. The first player to reach the **winning score (default: 100)** wins the game.  
6. Click **Reset** to start a new game.
